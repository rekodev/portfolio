# Portfolio

Personal portfolio built with Next.js. Focused on clean typography, responsive layout, and a minimal, modern UI.

## Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS
- shadcn/ui primitives
- next-themes

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run the dev server:

```bash
pnpm dev
```

Build for production:

```bash
pnpm build
```

Start the production server:

```bash
pnpm start
```

## Scripts

- `pnpm dev` - Start the development server
- `pnpm build` - Build the app
- `pnpm start` - Run the production build
- `pnpm lint` - Lint the codebase
